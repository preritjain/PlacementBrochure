Master of Science by Research

The M.S. by Research degree is intended for mature students who wish to learn and perform research in a supportive academic environment. It is awarded upon successful completion of a graduate-level research program, usually lasting four semesters. Beyond the satisfaction of a relatively small number of coursework requirements, the major focus of the program is on developing research skills,leading to the completion of a Master's thesis describing significant original results. Currently research is carried out in one of the following domains:

Computer Science

Mathematics & Basic Sciences

Electronic Systems Design

Networking,Communication & Signal Processing

Information Technology & Society

Software Engineering

    • Speech Guided Robots in Cyber-Physical Environment
      Human-Robot interaction through speech through the use of phone based isolated speech recognition for a small vocabulary.
    • MyAutobiography.in
      A commercial social networking website, specifically designed for institutions/organizations where users can update their autobiographies.
    • Bingya - Lost and Found
      Bingya is a social platform which uses string and image processing algorithms to match Lost and Found cases, currently implemented using Bing Maps, Windows Azure, Windows Phone and Windows 8.
    • Convolutional Neural Networks
      Designed a CNN (state-of-the-art image processing technique) from scratch in MATLAB to perform traffic sign recognition.
    • School Management System
      A web based application providing an online platform for teacher-student interaction with a user friendly interface.
      Tools used : PHP and MySQL
    • TagifyMe
      Developed a recommender system of hashtags for tweets on twitter, in collaboration with IIM - Lucknow. Tools Used - R, Python 2.7
    • IRB 120 Accuracy Tests
      Designed simple experiments to test the accuracy of the industrial grade robot IRB 120 (manufactured by ABB).
    • 4 LED Transmittance Multiparameter Wireless Pulse Oximeter
      Pulse oximeters are amongst the most widely used biomedical devices for monitoring the heart rate and blood pressure of patients. It generally uses two LEDs. This project achieved an upgrade in technology by using four LEDs for the calculation of multiple parameters like haemoglobin, spo2,heart rate, BP, etc. with better accuracy.
    • Lunar Rover Robot
      Designed and built a working model of a moon rover, with software to wirelessly control it and capture pictures and video. The team built a WiFi controlled vehicle using a Raspberry Pi board and a microcontroller, and assembled simple circuits for motors. Transfer of images and control involved Python programming.
    • NextWave
      Designed and prototyped an electronic panel to report cases of sexual harassment in public spaces. A collaborative effort between IIIT Bangalore, Fields of View (Bangalore) and MediaLAB Amsterdam.
      Tools used: Raspberry Pi, Python 2.7, TkInter GUI.
    • Indian Energy Game
      A commercial social networking website, specifically designed for institutions/organizations where users can update their autobiographies.
    • "Rubbish!”
      "Rubbish!" is a serious game that simulates Bangalore's current waste crisis, giving an insight to the economic and environmental constraints involved. 'Rubbish!' was developed as a collaboration between Fields of View, IIIT Bangalore and MediaLAB Amsterdam.
    • Bullet Train - Cost/Benefit Analysis
      Designed a schedule generating program in Python using current railway getting a bullet train versus upgrading the existing infrastructure.
