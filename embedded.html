    • Remote Water flow controller and Calculator
      A full fledged system (Industrially ready product) where user can give input through an App regarding the amount of Litres he wants and the Time at which the water has to be supplied. The system will take the input and accordingly On the motor at appropriate time. Alongwith controlling the flow of water, system also monitors how much litres of water has flown and at what rate the water has flown, which can be monitored through a website or App.
      Tools used:  Arduino, Raspberry Pi, Water Sensor, Android Studio.
    • Design of an ASIP(Application Specific Integrated Processor))
      "This project involves enhancing the instruction set of the VeSPA processor to enable newer functional units implementing Exponential, Trigonometric, Reciprocation, Square Root and the Inverse Square Root functions to be integrated into the existing architecture. This will involve development of these functional units, as well as, adding two new instructions to trigger computations in these functional units. The validation will be done by synthesizing the newer version of VeSPA on Xilinx FPGAs and then do a complete ASIC design of the same using the Cadence Design flow."
      Tools used:  XILINX ISE DESIGN SUITE ,CADENCE, VIS for verification purpose, SPARTAN , FPGA BOARD

    • IoT solution for LPG leakage detection
      One of the major security threat in the home is from LPG leakage. Under the concept of SmartHome, we built a smart and intelligent system which not only notifies hazardous level of LPG leakage to the mobile app but also takes care of home from LPG leakage by taking corrective actions. We also developed web version of the application so that user will be able to see what is happening to home irrespective of the platform.
      Tools used:  Raspberry Pi B , Arduino . Android Application , Google Cloud Messaging , REST Services in JAVA , D3.js for web data visualization
